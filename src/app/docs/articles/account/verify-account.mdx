# How to Verify Your Soraxi Account

Account verification is an important step to unlock all features of Soraxi. This guide will walk you through the simple verification process.

## Why Verify Your Account?

<Note type="info">
  Verification helps us ensure a secure marketplace for all users and unlocks
  these benefits:
</Note>

- ✓ **Full platform access** - Use all Soraxi features
- ✓ **Become a vendor** - Create your own storefront
- ✓ **Enhanced security** - Protect your account and funds
- ✓ **Trust badge** - Build credibility with other users

## Step-by-Step Verification Process

### Step 1: Access Your Profile

1. **Locate your username** at the top right-hand side of any Soraxi page
2. **Click on your username** to open the dropdown menu
3. **Select "Profile"** from the menu options

<Screenshot
  src="/docs-images/account/soraxi-verification-step-1.png"
  alt="User profile dropdown menu"
  caption="Click your username and select 'Profile' from the dropdown"
/>

<Warning>
  If you don't see your username, you need to sign in first. Click "Sign In" and
  enter your credentials.
</Warning>

### Step 2: Start Verification

1. On your profile page, look for the **"Account Verification"** section
2. You'll see a green banner with a verification prompt
3. Click the **"Complete Verification"** button

<Screenshot
  src="/docs-images/account/soraxi-verification-step-2.png"
  alt="Profile page verification section"
  caption="Click 'Complete Verification' in the Account Verification section"
/>

<Tip>
  The verification section only appears if your account isn't verified. If you
  don't see it, your account might already be verified!
</Tip>

### Step 3: Enter Verification Code

You'll be redirected to the verification page where you need to:

1. **Check your email** for the 6-digit verification code
2. **Enter the code** in the six input boxes
3. **Click "Verify Account"** to complete the process

<Screenshot
  src="/docs-images/account/soraxi-verification-step-3.png"
  alt="OTP verification screen"
  caption="Enter the 6-digit code sent to your email address"
/>

## Understanding the Verification Interface

### OTP Input Fields

The verification form includes:

- **Six input boxes** - One for each digit of your code
- **Real-time validation** - Automatically moves to next box
- **Clear visual feedback** - Highlights active and completed fields

### Action Buttons

- **Verify Account** - Submits your code for verification
- **Resend Code** - Requests a new code if needed

<Note type="success">
  The verification code is automatically sent to the email address you used
  during sign-up.
</Note>

## Common Verification Scenarios

### I Didn't Receive the Code

**If you don't see the verification email:**

1. **Wait 2-3 minutes** - Emails can take a moment to arrive
2. **Check spam/junk folder** - Sometimes emails get filtered
3. **Verify email address** - Ensure you used the correct email
4. **Click "Resend Code"** - Request a new verification code

<Screenshot
  src="/docs-images/account/soraxi-verification-step-4.png"
  alt="Resend code button"
  caption="Use the 'Resend Code' link if you didn't receive the email"
/>

### The Code Expired

**Verification codes expire after 15 minutes. If your code isn't working:**

1. Click **"Resend Code"** to get a new one
2. Enter the new code immediately
3. Complete verification within 15 minutes

### I Entered the Wrong Code

**If you enter an incorrect code:**

1. You'll see an error message
2. The input boxes will clear automatically
3. You can enter the correct code or request a new one

## After Successful Verification

Once your account is verified:

1. **Automatic redirect** - You'll be taken back to your profile
2. **Verification badge** - Your profile will show "Verified Account"
3. **Full access granted** - All platform features are now available

## Troubleshooting Common Issues

### "Verification Failed" Error

**If you see a verification failed message:**

- Ensure you entered all 6 digits correctly
- Check that the code hasn't expired (request a new one if needed)
- Make sure you're using the most recent code sent to your email

### "Resend Code" Button is Disabled

**The resend button has a 60-second cooldown:**

- Wait for the countdown timer to reach zero
- The button will automatically become available
- This prevents email spam and ensures delivery

### Page Not Loading

**If the verification page isn't loading properly:**

- Refresh your browser
- Clear your browser cache and cookies
- Try using a different browser
- Ensure you have a stable internet connection

## Security Tips

<Warning>
  Never share your verification code with anyone. Soraxi will never ask for your
  code via phone, email, or chat.
</Warning>

- **Keep codes private** - Treat them like passwords
- **Use secure networks** - Avoid public Wi-Fi for verification
- **Verify email security** - Ensure your email account is protected
- **Log out from shared devices** - Always sign out after use

## FAQs

**Q: How long does verification take?**
A: The email verification process is instant. Once you enter the correct code, your account is verified immediately.

**Q: Can I verify without an email?**
A: Currently, email verification is required. Ensure you have access to the email used during sign-up.

**Q: What if I changed my email address?**
A: You need to update your email in account settings first, then request a new verification code.

**Q: Is verification mandatory?**
A: While you can browse without verification, full platform access require a verified account.

**Q: How long is the verification code valid?**
A: Verification codes expire after 15 minutes for security reasons.
